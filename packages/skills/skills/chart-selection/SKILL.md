# 图表选型决策树

## 按问题选型(不是按好看选型)
- 分类间比较 → 柱状图;类别名长/类别多(>8)→ 水平条形并按值排序
- 时间趋势 → 折线图;多序列 ≤5 条,再多就拆分面板
- 构成占比 → 饼/环图(≤6 类,首类从 12 点方向起);>6 类 → 排序条形图
- 构成随时间变化 → 堆叠面积/堆叠柱
- 两变量相关 → 散点图(可加趋势线);分布 → 直方图/箱线
- 目标完成度 → 子弹图/进度条,别用仪表盘浪费空间

## 规范(专业底线)
- 柱状图数值轴必须从 0 起(截断轴会夸大差异);折线图可不从 0 但要标明
- 不用 3D、不用双 Y 轴误导(确需双轴,两轴刻度要可比且分色标注)
- 颜色 ≤6 且色盲友好;强调色只给要说的那一根,其余灰
- 标题写结论("华东区 Q2 销量领先 32%")而非维度名("各区销量")
- 排序要有含义:条形按值排、时间按时间排,别按字母排

## 反例
- 饼图切 12 类、彩虹配色、图例遮数据
- 用面积/气泡大小编码精确比较(人眼对面积不敏感)
