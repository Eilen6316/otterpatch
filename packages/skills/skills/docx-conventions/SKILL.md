# Word 文档专业惯例(通用)

## 结构必须"真"
- 标题必须套真标题样式(block=h1/h2/h3)——手动放大加粗的"假标题"进不了导航窗格、抓不进自动目录、样式不可维护;规范化的第一刀永远是把假标题转真
- 目录依赖真标题:用户要目录时,先把各级标题 block 套齐,再告知用「引用→目录」自动生成即可,不要手打目录文本
- 列表要用真编号/项目符号(编辑器的列表能力),不要手打"1、""•"字符冒充——真列表续项自动编号、缩进统一
- 层级不越级(h1 后别直接 h3);同级标题字体字号必须一致

## 修订最小化(外科原则)
- 改写一句话时 quote 只圈**真正变化的部分**,前后不变的词不要卷进来——修订痕迹越小,审阅者越容易看懂、接受后残留风险越低
- 改文字与改格式拆成两条 edit,各自可独立接受/拒绝
- 同一段被多处修改时按出现顺序分条,别一条 edit 吞整段

## 标点与排印底线
- 中文正文全角标点(,;。而非 , ; .);中英文/中数字之间补半角空格("使用 Word 2021")
- 引号用弯引号("" '')、省略号用……、破折号用——;直引号/三个点/两个减号是排印硬伤
- 段落间距用样式/行距表达,不要用连续空段撑距离;文档里连续 2+ 空段是清理对象

## 页面与单位常识
- A4 纵向是中文文档默认;页面级改动(分栏/边距/方向)作用于整篇,须 all=true 下发
- 字号对照:初号42 / 小初36 / 一号26 / 二号22 / 三号16 / 四号14 / 小四12 / 五号10.5(pt)
- 行距:正文 1.5 倍是通用安全值;公文场景以 GB/T 9704 手册为准

## 反例
- 用空格居中标题(要 align=center);用空行造分页(分页是页面属性)
- "顺手"把用户的自定义样式体系改成默认样式
- 为了省事把整段替换掉,只为改其中三个字
